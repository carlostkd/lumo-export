Help & Guide

📥 Exporting Chats

One-Click Export: Select messages (or use "Select All") and click Export Selected.

Formats: Choose from JSON (structured), Plain Text, HTML (visual), or Markdown (preserves code blocks).

Encryption: Check "Encrypt with Password" to secure your export with AES-256.

🔍 Finding & Replacing Text

Enable Find & Replace to anonymize or modify content before export.

Pro Tip: Enter multiple words separated by commas!

  • Find: Save,Home,John
  • Replace: Downloaded,Work,Justin

This will replace "Save"→"Downloaded", "Home"→"Work", and "John"→"Justin" simultaneously.

📦 Extracting Code Blocks

Click 📦 Extract Code Blocks to scan the entire chat for code snippets.

Preview all snippets, select specific ones, and export them as a single Markdown file or individual text files.

🛡️ Security & Integrity

Environment Health: Checks if your browser supports necessary APIs.

Extension Audit: Scans for other extensions that might conflict or pose risks.

Code Integrity: Verifies that the extension code hasn't been tampered with using SHA-256 hashing.

💡 Tips & Tricks

  • Resizable Window: Click the ⛶ button to expand the popup for better visibility.
  • Smart Tooltips: Hover over any message to read the full text without selecting it.
  • Live Search: Type in the search box to instantly filter messages and highlight matches.
  • Custom Dropdown: Click the format selector to see all export options in a styled menu.
